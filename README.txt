*HƯỚNG DẪN KHỞI CHẠY TRANG WEB*

- Bước 1: Chạy lệnh "npm install" để tiến hành cài đặt file node_modules tại thư mục WebSocket
- Bước 2: Truy cập vào thư mục server và tiếp tục chạy lệnh "npm install" để cài đặt file node_modules tại thư mục này
- Bước 3: Tại thư mục server chạy lệnh "npm run dev" để tiến hành khởi chạy server
- Bước 4: Trở lại thư mục WebSocket và chạy câu lệnh "npm run serve" để tiến hành khởi chạy client 
- Bước 5: Sau khi khởi chạy client hoàn tất thì truy cập vào đường link hiển thị ở cửa sổ TERMINAL để
truy cập đến trang web
